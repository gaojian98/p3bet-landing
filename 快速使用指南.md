## 🎨 常见修改场景

### 场景1：添加新游戏

```javascript
// 在 config.js 中找到对应分类
{
    name: 'Trò chơi nổi bật',
    games: [
        // 现有游戏...
        
        // 添加新游戏
        { 
            image: 'assets/images/games/new-game.png', 
            name: '新游戏名称', 
            hot: true  // 显示HOT标签
        }
    ]
}
```

### 场景2：添加新分类

```javascript
// 在 CONFIG_GAME_CATEGORIES 数组末尾添加
{
    name: '我的新分类',
    games: [
        { image: 'assets/images/games/game1.png', name: '游戏1' },
        { image: 'assets/images/games/game2.png', name: '游戏2', hot: true }
    ]
}
```

### 场景3：添加新视频

```javascript
// 在 CONFIG_VIDEOS 数组中添加
{
    id: 'video3',                           // 唯一ID
    source: 'assets/videos/video3.mp4',     // 视频文件
    title: '新视频标题',                     // 显示标题
    cover: 'assets/images/videos/cover3.jpg' // 封面图片
}
```

### 场景4：删除游戏

直接删除 `games` 数组中对应的游戏对象即可。

### 场景5：修改颜色主题

编辑 `styles.css` 的 CSS 变量：

```css
:root {
    --primary-color: #FF6B00;      /* 改成你的主色 */
    --secondary-color: #1a237e;    /* 改成你的副色 */
}
```

## 🐛 常见问题解决

### 问题1：图片不显示
**原因**：路径错误或文件不存在
**解决**：
- 检查 `config.js` 中的路径
- 确认文件名大小写正确
- 确保图片文件存在

### 问题2：视频无法播放
**原因**：格式不支持或文件过大
**解决**：
- 转换为 MP4 格式
- 压缩视频文件
- 检查浏览器控制台错误

### 问题3：手机上样式错乱
**原因**：缓存或兼容性问题
**解决**：
- 清除浏览器缓存
- 使用无痕模式测试
- 检查 viewport 设置

### 问题4：轮播图不自动播放
**原因**：JavaScript 错误或配置问题
**解决**：
- 打开浏览器控制台查看错误
- 检查 `config.js` 语法
- 确认 `app.js` 已加载
